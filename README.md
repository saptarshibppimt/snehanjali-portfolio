# Snehanjali Das — Personal Portfolio

A beautifully designed, single-page personal portfolio website showcasing the academic achievements, creative work, and professional journey of **Snehanjali Das** — Educator, Author, Hindustani Classical Vocalist, and UGC-NET / MH-SET qualified scholar.

## 🌟 Live Site
> Hosted via [Vercel](https://vercel.com) — link will appear after deployment

## 🛠️ Tech Stack
- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom properties, animations, glassmorphism, responsive grid
- **Vanilla JavaScript** — Intersection Observer, counter animations, particle effects, form handling

## ✨ Features
- Animated particle hero section with 3D avatar tilt
- Live animated stat counters (UGC-NET percentile, CGPA, publications...)
- Scrolling timeline for education
- Animated skill bars
- Interactive card tilt effects (desktop)
- Smooth reveal animations on scroll
- Mobile-first responsive design
- Accessible navigation with hamburger menu
- Contact form with validation
- Section active-link highlighting on scroll

## 🚀 Deploy to Vercel

1. Push this repository to GitHub
2. Go to [vercel.com](https://vercel.com) and sign in with GitHub
3. Click **"Add New Project"** → Import this repository
4. Framework: **Other** (no build step needed)
5. Root Directory: `./`
6. Click **Deploy** — done! ✅
